# Customer Churn Prediction & Analytics

## 📌 Overview
This project predicts customer churn using machine learning and provides business insights using Power BI.

## 📊 Dataset
- Customer data
- Usage data
- Churn labels

## ⚙️ Tech Stack
- Python (Pandas, NumPy, Scikit-learn, XGBoost)
- Jupyter Notebook
- Power BI

## 🚀 Key Features
- Data cleaning & EDA
- Feature engineering
- Churn prediction model
- Interactive dashboard (Power BI)

## 📈 Results
- Achieved high recall and precision for churn detection
- Identified high-risk segments (e.g., month-to-month users)
